# Call Center Workload Forecasting and Sentiment Analysis

## Project Overview
This project predicts future call volumes and analyzes customer sentiment for a call center.  
It helps managers make data-driven decisions for staffing and improving customer service.

---

## Features
- Display **historical call volumes**  
- Predict **future call volumes** using ML models  
- Analyze **customer sentiment** (positive/negative)  
- Interactive **Streamlit dashboard** for easy visualization  
- Developed and tested using **Google Colab** for ML and Python processing

---

## Google Technologies Used
- **Google Colab**: For running and training the ML models and processing data  

---

## How It Solves the Problem
- Enables call center managers to **anticipate high call volumes** and plan staff accordingly  
- Tracks **customer sentiment trends** to improve service quality  
- Combines all insights in a **single interactive dashboard**
